# kube-cloud-diagnostics
A docker image helpful in diagnostics regarding kubernetes operations, forensic analysis, or penetrtion testing. It can be useful either locally or when deployed as a pod.

This project is still rather new so the list of included tools is still sparse, but look forward to more being included in the future. 

## Tools provided 
* kubeeye
* nmap 
* bind9-dnsutils 
* netcat 
* socat 
* nginx 
* openssl 
* binwalk 
* nmap 
* curl
